# BROKE Website

## Project Overview
BROKE is a streetwear fashion website built with HTML and CSS. This project showcases a brand-focused homepage, product shop pages, responsive navigation, and improved page structure.

## Project Files
- `Homepage.html`
- `ABOUT US .html`
- `CONTACT.html`
- `SIGN IN .html`
- `Shop.html`, `Shop1.html`, `Shop2.html`, `Shop3.html`
- `css/BROKE.css`
- `images/`
- `javascript/BROKE.js`

## Usage
1. Open the project folder in your code editor.
2. Open any `.html` file in your browser.
3. Edit page content or update `css/BROKE.css` for custom styling.

## Design Notes
- Modernized the existing layout without changing the original brand feel.
- Improved responsive behavior for mobile and tablet screens.
- Kept the current gray/nav accent palette and simple product presentation.

## Recommended Updates
- Add JavaScript functionality for search and lightbox interactivity.
- Replace placeholder images with final product assets.
- Rename files without spaces for more reliable linking, e.g. `ABOUT US .html` to `about-us.html`.

## Current Features
- Multi-page HTML website
- External CSS styling
- Responsive navigation
- Product gallery layout
- Contact page with map support
- Video and sign-in form pages
